# Stacks Donation DApp

A beginner-friendly Stacks DApp that allows users to connect their wallet and donate STX to a smart contract.

## Features
- Wallet connection using @stacks/connect
- Smart contract calls using @stacks/transactions
- Clarity smart contract for storing donations
- Simple React frontend

## Setup

```bash
cd frontend
npm install
npm run dev
